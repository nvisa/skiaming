注意：

本示例在 Microsoft Office Excel 2007 及更高版本中运行。

此示例演示如何从关系数据库或 XML 文件中检索数据，并在 Excel 中使用该数据。

该示例是一个订单履行工作表。 工作表中显示了客户订单的详细信息，以便于将货品打包发运。 工作表还显示了各种货品的当前库存，以使用户了解订单可否履行。

运行示例

  1. 按 F5。

  2. 在“Select a Customer”（选择客户）列表中选择客户名称。

    “未履行的订单”列表便会被那些未标记为“已履行”的订单的订单编号所填充。

  3. 在“未履行的订单”列表中选择一个编号。

    工作表便会被该订单的详细信息所填充。

  4. 在“指示订单状态”列表中，单击“已履行”。

    当状态设置为“已履行”后：

    • 订单状态被保存回内存中的数据集中。

    • 对于订单中所订购的每种产品，都会从内存中的数据集中该产品的“UnitsInStock”元素中减去该产品的订购数量。 这样可以确保每种产品的正确“UnitsInStock”值可用于下一次订购。

    • 订单从“Unfulfilled Orders”（未履行的订单）列表中移除。

本示例演示如何：

  • 检索和使用从 SQL Server 数据库提取的 XML 文件中的数据。

  • 使用数据绑定的 Excel 控件显示数据。

  • 处理多组主/从数据。
