注意：

本示例在 Microsoft Office Word 2007 及更高版本中运行。

此示例演示如何将 Word 文档中的内容控件绑定到 Microsoft Office Access 数据库中的字段。 文档中的每个内容控件绑定到 Northwind 示例数据库中的一个不同字段。 文档还显示一个操作窗格，可以用来在雇员记录中导航、搜索特定的雇员记录以及将在一个内容控件中所做的更改保存到该数据库中。

运行此示例

  1. 按 F5。

    确认表中的内容控件是否显示关于名为 Nancy Davolio 的雇员的信息。

  2. 在操作窗格中，单击“Next”（下一个）和“Previous”（上一个）按钮查看不同的雇员记录。

    确认内容控件是否显示来自当前所选雇员记录的信息。

  3. 在操作窗格中，在“Input ID”（输入 ID）文本框中输入一个 1 到 9 之间的数字，然后单击“Search by ID”（按 ID 搜索）按钮。

    确认内容控件是否显示来自与该 ID 匹配的雇员记录的信息。

  4. 在“Title”（职称）单元格右侧的单元格中单击，为当前雇员键入一个新职称。

  5. 在操作窗格中，单击“Submit change to Title”（提交对职称的更改）按钮将新职称保存到数据库中。

  6. 在 Access 中打开 Northwind.mdb 数据库，确认该雇员的“Title”（职称）字段是否具有该新职称。

要求

此示例需要以下应用程序：

  • Visual Studio Tools for Office。

  • Microsoft Office Word 2007。

  • Microsoft Office Access。

  • 用于 Microsoft Office Access 的 Northwind 示例数据库（包含在示例中）。

演示

本示例演示以下概念：

  • 将以下内容控件绑定到数据库字段：

    • PlainTextContentControl

    • PictureContentControl

    • DatePickerContentControl

  • 将数据绑定内容控件中的数据保存到数据库中（这也称为双向数据绑定）。

  • 使用操作窗格。
