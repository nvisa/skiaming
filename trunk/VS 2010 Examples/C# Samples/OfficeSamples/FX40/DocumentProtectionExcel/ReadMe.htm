注意：

本示例在 Microsoft Office Excel 2007 及更高版本中运行。

本示例演示可用来保护工作表的结构组件的各种技术。 这些技术可以帮助防止用户意外删除控件或在其他情况下更改解决方案文档的重要部分。

运行此示例

  1. 按 F5。

  2. 按照 Sheet1 和 Sheet2 上的说明操作。

演示

  • 当用户删除控件后如何重新创建该控件。

  • 如何向受保护的工作表上的数据绑定 ListObject 控件写入数据。

  • 如何向受保护的工作表上的 NamedRange 控件写入数据。

  • 如何保护 NamedRange 控件中的值，避免未经授权的修改。

  • 如何保护 Windows 窗体控件，使得用户不能修改或删除它，但是您的代码可以访问它。
