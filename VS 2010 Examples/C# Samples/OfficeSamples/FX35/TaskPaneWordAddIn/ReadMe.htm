注意：

本示例在 Microsoft Office Word 2007 及更高版本中运行。

此示例演示如何为 Microsoft Office Word 创建自定义任务窗格。 该任务窗格包含一个文本框和一个按钮。 当用户在文本框中键入文本并单击按钮时，文本框中的字符串会添加到活动文档的开头。

当加载外接程序时，此示例会自动显示该任务窗格。 但是，此示例不为用户提供显示或隐藏该任务窗格的方法。 通常情况下，外接程序应该提供 UI 元素（例如功能区上的按钮），用户可以单击该元素来显示或隐藏任务窗格。 有关演示如何执行此操作的演练，请参见“演练：将自定义任务窗格与功能区按钮同步”。

为创建自定义任务窗格，此示例使用 Visual Studio Tools for Office 提供的 CustomTaskPane 和 CustomTaskPaneCollection 类。 有关使用这些类来创建自定义任务窗格的更多信息，请参见“自定义任务窗格概述”。

运行此示例

  1. 按 F5。

  2. 在“Contoso”任务窗格中的文本框中键入一些文本。

  3. 单击“插入”按钮。

    验证在文本框中键入的文本是否插入到活动文档的开头。

要求

此示例需要以下应用程序：

  • Visual Studio Tools for Office。

  • Microsoft Office Word 2007。

演示

本示例演示以下概念：

  • 在 Word 外接程序中创建和显示自定义任务窗格。

  • 将在自定义任务窗格中输入的文本插入到活动 Word 文档中。
